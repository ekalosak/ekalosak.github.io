Introduction

SHAPE reactivity experiments probe the structure of nucleic acid products, usually RNA. To do so, a binding agent (in this case SHAPE) is introduced into a solution containing the target RNA molecule. Ideally, SHAPE binds with higher affinity to more exposed, non-paired portions of the molecule (e.g. hairpin loops) and with less affinity to paired portions of the molecule (e.g. helices). The following analysis esposes some of the raw data produced by such a SHAPE reactivity experiment, examines coverage, and calculates differential reactivity using two methods.

Reactivity

In this section I present two methods for calculating reactivity and apply them both to the pT181 dataset.

Aviran et al. 2011

\[ \beta_k^{(A)} = \frac{ \frac{x_k}{\sum_{i=k}^{n+1}{x_i}} - \frac{y_k}{\sum_{i=k}^{n+1}{y_i}} }{ 1 - \frac{y_k}{\sum_{i=k}^{n+1}{y_i}} } \vee 0 \]

Ding et al. 2014

\[ \beta_k^{(D)} = \frac{\log(1-x_k)}{\frac{1}{n} \sum_{i=1}^{n+1} \log(1+x_i)} - \frac{\log(1-y_k)}{\frac{1}{n} \sum_{i=1}^{n+1} \log(1+y_i)} \vee 0 \]

## @knitr setup

rm(list=ls())

library(ggplot2)
library(reshape)

d = read.table("./pT181_Sense_112_adducts.txt", header=T)

## @knitr reactivity

x = c(d$treated_mods[-1],
      d$treated_mods[1])
y = c(d$untreated_mods[-1],
      d$untreated_mods[1])
n = length(x) - 1
stopifnot(length(x) == length(y))

ba = function(k) {
    # Reactivity calculation from Aviran et al. 2011
    stopifnot(k>=0)

    a = x[k] / sum(x[-(1:k)])
    b = y[k] / sum(y[-(1:k)])
    numer = a-b
    denom = 1-b
    r = numer/denom
    if(is.nan(r)){return(0)}
    return(max(r,0))
}

bd = function(k) {
    # Reactivity calculation from Ding et al. 2014
    stopifnot(k>=0)

    na = log(1+x[k])
    da = (1/n)*sum(log(1+x))
    nb = log(1+y[k])
    db = (1/n)*sum(log(1+y))
    a = na/da
    b = nb/db
    r = a-b
    if(is.nan(r)){return(0)}
    return(max(r,0))
}

n = dim(d)[1] - 1
reac = data.frame(
        Aviran = sapply(1:n, ba),
        Ding = sapply(1:n, bd),
        Nucleotide = d$nucleotide[-1],
        five_prime_offset = d$five_prime_offset[-1]
    )

reacp = melt(reac, id=c("Nucleotide", "five_prime_offset"))
levels(reacp$variable) = c("Aviran et al. 2011", "Ding et al. 2014")
p1 = ggplot(reacp, aes(x=five_prime_offset, y=value)) +
    geom_histogram(aes(fill=Nucleotide), stat="identity") +
    geom_freqpoly(stat="identity") +
    facet_wrap(~variable, nrow=2) +
    labs(
        title="SHAPE reactivity on pT181 plasmid using two calculation methods",
        x="5' offset", y="Signal strength")
