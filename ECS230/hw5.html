Brief proof of a property of eigenvalues

Let \(A\) be a real \(n\times n\) matrix and \(\lambda\) an eigenvalue of \(A\). Here, I show that \(\lambda\) is an eigenvalue of \(A^\top\) as well.

Notice that the eigenvalues of \(A\) are the roots of its characteristic polynomial \(f(\lambda) = det(A-\lambda I)\). Similarly, let \(f^\top(\lambda) = det(A^\top-\lambda I)\) be the characteristic polynomial of the transposed matrix \(A^\top\). This proof boils down to showing that \(f(\lambda) = f^\top(\lambda)\) as follows:

  1. \(f(\lambda) = det(A-\lambda I)\)
  2. \(det(A-\lambda I) = det((A-\lambda I)^\top)\)
  3. \(det((A-\lambda I)^\top) = det(A^\top-\lambda I^\top)\)
  4. \(det(A^\top-\lambda I^\top) = det(A^\top - \lambda I)\)
  5. \(det(A^\top - \lambda I) = f^\top(\lambda) \quad \blacksquare\)

Warrant for each of the steps in the proof above follow:

The first and final steps of the proof are the respective definitions of \(f(\lambda)\) and \(f^\top(\lambda)\). The second step can be seen by expanding \(A-\lambda I\) into its \(QR\) decomposition and using determinant product rules. The third and fourth steps are obvious.

Power method

To solve for the dominant eigenvalue of a matrix \(A\), the power method can be used. In short, the method is as follows: \[b_{k+1} = \frac{Ab_k}{||Ab_k||}\] This method and several extensions are implemented in the source code included at the end of this report.

Reproducing a simple result

Using the graph presented in figure 1 of Bryan and Leise (2006), the program produes the following vector as the dominant L1-normalized eigenvector: \[v_1 = \left( \begin{matrix} 0.387097 \\ 0.129032 \\ 0.290323 \\ 0.193548 \end{matrix} \right) \] This solution is produced using the “bryan_leise_2006_fig1.dat” input file and corresponds to the ranking \(A, C, D, B\).

A PageRank manipulation strategy

Using the strategy described in exercise 1 of Bryan and Leise where the owners of page \(C\) make a new page \(E\) and link both together, the dominant eigenvector becomes \[v_2 = \left( \begin{matrix} 0.244898 \\ 0.081633 \\ 0.367347 \\ 0.122449 \\ 0.183673 \end{matrix} \right) \] This new dominant eigenvector \(v_2\) corresponds to the ranking \(C, A, E, D, B\). Notice that, using this strategy, the owners of page \(3\) were able to manipulate the ranks.

A chain graph and the shifted power method

Using a chain graph of length \(n=4\) and an initial vector \[b_0 = \left( \begin{matrix} 1.000000 \\ 1.000000 \\ 1.000000 \\ 1.000000 \end{matrix} \right) \]

my program converges to the dominant eigenvector \[v_3 = \left( \begin{matrix} 0.166667 \\ 0.333333 \\ 0.333333 \\ 0.166667 \end{matrix} \right) \]

corresponding to the ranking \(\{2, 3\}, \{1, 4\}\) as expected. However, due to the fact that the spectrum of the induced matrix column-stochastic link matrix \(A\) is \[ spec(A) = spec( \left( \begin{matrix} 0 & 1/2 & 0 & 0 \\ 1 & 0 & 1/2 & 0 \\ 0 & 1/2 & 0 & 1 \\ 0 & 0 & 1/2 & 0 \end{matrix} \right) ) = \{1, \frac{1}{2}, \frac{-1}{2}, -1\} \]

yielding two dominant eigenvectors corresponding to the eigenvalues \(1\) and \(-1\), the power method fails to converge for impropperly selected initial vectors. For example, when using \[b_0 = \left( \begin{matrix} 1.00000 \\ 2.00000 \\ 3.00000 \\ 4.00000 \end{matrix} \right) \]

the result oscilates between \[ \left( \begin{matrix} 0.20000 \\ 0.26666 \\ 0.40000 \\ 0.13333 \end{matrix} \right) \textrm{ and } \left( \begin{matrix} 0.372104 \\ 0.496139 \\ 0.744208 \\ 0.248069 \end{matrix} \right) \]

To solve this problem of non-convergence, I implement the shift method. In short, this replaces the matrix \(A\) with \(B=A+cI\) for some constant \(c\). Note that the eigenvectors are identical for \(A\) and \(B\): if \(v\) is an eigenvector of \(A\), then \(Bv=(A+cI)v=Av+cv=(\lambda+c)v\). Here, noting that the spectrum is bounded by \(-1\) and \(1\), I use \(c=\frac{1}{4}\) to force the desired dominant eigenvector to have an eigenvalue of \(1.25\) and bring the other eigenvectors away from dominance.

Upon implementation, the shifted power method converges for all tested initial vectors.

Shifted power method vs. regularized power method

The shifted power method has already been discussed, so I’ll move forward assuming the reader understands it. The regularized power method is only slightly different, and I seek to illuminate the difference as follows: wheras; with the shifted method, the matrix \(cI\) was added to \(A\) in the form \(B = A+cI\); in the regularized method, the matrix \(\frac{1}{n}1_{n\times n}\) is added to \(A\) in \(B^\star=A+\frac{1}{n}1_{n\times n}\).

The merit of the shifted method is that the eigenvectors are invariant under this type of shift. The merit of the regularized method is that it may ensure the matrix \(B^\star\) is well conditioned when data are limited and it guarantees a fully connected web.

The ranking of a linear chain would be invariant using either of the two methods discussed above. However, the regularized power method may suffer from underflow as \(n\rightarrow\infty\). The shifted power method does not suffer from such an issue.

int main(int argc, char** argv)
{
    // read data into memory from disk
    ifp = fopen(data_fn, mode);
    if (ifp == NULL) {
        fprintf(stderr, "Cannot open file\n"); // error if it can't open
        exit(1);
    }

    n = 1;
    firstline = 1;
    double *A = (double*) malloc(n*n* sizeof(double)); // placeholder malloc
    double *b1 = (double*) malloc(n* sizeof(double)); // placeholder malloc
    double *b2 = (double*) malloc(n* sizeof(double)); // placeholder malloc

    // read file
    char * line = NULL;
    size_t len = 0;
    ssize_t read;
    char * pch;
    i = 0;
    while ((read = getline(&line, &len, ifp)) != -1){
        if(firstline == 1){

            /* printf("Read first line (len %zu) : ", read); */
            /* printf("%s\n", line); */
            n = atoi(line); // n_obs is first line of data.dat
            firstline = 0;
            /* printf("Using %d observations\n", n); */
            printf("n=%d\n", n);

            // allocate memory for the input data
            // NOTE: A is col-major indexed i.e. A[i + n*j] = A_(i,j)
            A = realloc(A, n*n*sizeof(double));
            b1 = realloc(b1, n*sizeof(double));
            b2 = realloc(b2, n*sizeof(double));

            if (A == NULL) {
                fprintf(stderr, "malloc failed\n");
                return(1);
            }else{
                /* printf("Memory allocated for input and design matrices\n\n"); */
            }

            // initialize A as 0s, b as arbitrary
            int i1, i2;
            for(i1=0;i1<n;i1++){
                for(i2=0;i2<n;i2++){
                    A[i1 + n*i2] = 0.0;
                    /* printf(A[i + n*j]); */
                }
                b1[i1] = (double) i1 + 1;//1.0;///n;
                /* b1[i1] = 1.0; */
                b2[i1] = b1[i1];
            }

        }else{

            // Calculate and install input into matrix A
            /* printf("Read line (len %zu) : ", read); */
            /* printf("%s", line); */
            pch = strtok(line," "); // parse line into matrix A's j'th col
            k = 0;
            firstline = 1;
            double nj;
            while (pch != NULL) {
                if(firstline == 1){
                    nj = atof(pch); // number of outlinks for node j
                    firstline = 0;
                    /* printf("Col %i connected to %d rows\n", i, (int) nj); */
                }else{
                    k = atof(pch); // node pointed to by j is k
                    /* printf("k,i:%d,%d:\n", k,i); */
                    A[k-1 + n*i] = 1.0/nj;
                    /* printf("A_%d,%d: %f\n", i, k-1, A[i + n*(k-1)]); */
                }

                pch = strtok(NULL, " ");
            }

            i++;
            /* printf("\n"); */
        }
    }
    fclose(ifp);
    printf("\n");

    // print A
    i = 0;
    j = 0;
    printf("A:\n");
    for(i=0;i<n;i++){
        for(j=0;j<n;j++){
            printf("%f ", A[i + n*j]);
        }
        printf("\n");
    }

    // normalize b_0
    double norm_b;
    int INCX = 1;
    norm_b = cblas_dnrm2(n, b1, INCX);
    /* printf("\nnorm b:\n%f\tnormalizing...\n", norm_b); */
    for(i=0; i<n; i++){
        b1[i] = b1[i] / norm_b;
    }
    norm_b = cblas_dnrm2(n, b1, INCX);

    // print normalized b_0
    i = 0;
    printf("\nb0:\n");
    for(i=0;i<n;i++){
        printf("%f\n", b1[i]);
    }
    printf("\n");
    /* printf("\nnorm b:\n%f\n", norm_b); */

    // Power method (shifted)
    // $b_{k+1} = A b_k \over \norm{A b_k}$ until convergence, determined as
    // follows: $\infty_norm{ b_{k+1} - b_k } < \epsilon = 1.0e-6$
    //
    // Using dgemv_(A, b)

    double shift = 0.25; // for shifted power method
    for(i=0; i<n; i++){
        A[i + n*i] = A[i + n*i] + shift; // add shift*I to A
    }

    TRANS = 'N'; // transpose the matrix A
    M = n; // rows of A (number of obs)
    N = n; // columns of A (degree of polynom)
    LDA = n; // leading dimension of A
    INCB = 1;
    ALPHA = 1.0;
    BETA = 0.0;

    double epsilon = 1.0e-6;
    /* double epsilon = 1.0e-2; */
    double delta = 1.0;
    int breaker = 1e2;
    itt = 0;
    while(delta > epsilon){

        // copy b1 <- b2
        for(i=0;i<n;i++){
            b1[i] = b2[i];
            /* printf("%f\n", b1[i]); */
        }

        // normalize b1
        norm_b = cblas_dnrm2(n, b1, INCX);
        for(i=0; i<n; i++){
            b1[i] = b1[i] / norm_b;
        }
        /* printf("\nnorm b%d:%f\n", itt, norm_b); */

        // multiply b2 <- A b1
        dgemv_(&TRANS, &M, &N, &ALPHA,
                A, &LDA,
                b1, &INCB, &BETA,
                b2, &INCB);

        // normalize b2
        norm_b = cblas_dnrm2(n, b2, INCX);
        /* printf("\nb%d:\n", itt); */
        for(i=0;i<n;i++){
            b2[i] = b2[i] / norm_b;
            /* printf("%f\n", b2[i]); */
        }

        // calculate delta
        delta = 0.0;
        for(i=0; i<n; i++){
            delta = delta + fabs(b1[i] - b2[i]);
        }
        printf("delta %d: %f\n", itt, delta);

        itt++;

        if(itt > breaker){
            printf("\nNot converging fast enough\n");
            printf("breaking...\n");
            break;
        }
    }

    // scale eigenvector so L1 norm is 1
    norm_b = cblas_dasum(n, b1, 1);
    for(i=0; i<n; i++){
        b1[i] = b1[i] / norm_b;
    }

    if(itt > breaker){
        printf("\nFailed to converge\n");
        printf("After %d iterations to vectors:\n", itt);
        for(i=0; i<n; i++){
            printf("%f\n", b1[i]);
        }
        printf("and\n");
        for(i=0; i<n; i++){
            printf("%f\n", b2[i]);
        }
    }else{
        printf("\nConverged successfully\n");
        printf("After %d iterations to dominant eigenvector b:\n", itt);
        for(i=0; i<n; i++){
            printf("%f\n", b1[i]);
        }
    }

    return 0;
}

a2[2,1] = 1;
a2[2,3] = 1;
a2[3,2] = 1;
a2[3,4] = 1;
a2[4,3] = 1;
n2 = network(a2, directed=T)
plot.network(n2, label=c("A", "B", "C", "D"),
             arrowhead.cex=2, label.cex=2, vertex.cex=2,
             vertex.col="red")
title("Figure 3 - a chain graph")
